WellPaw: Dog Monitoring System (DMS) is an AI-driven digital diagnostic assistant designed to bridge the gap between pet owners and veterinary professionals. By integrating clinical questionnaires with real-time video analysis, WellPaw provides objective data to assist staff during animal consultations.

🚀 Core Features
Behavioral AI Analysis: Utilizes YOLOv5 to detect physical stress indicators and postural markers in dog video footage.

Clinical Workflow Integration: Consolidates owner-submitted data with professional veterinary diagnostic indicators.

Automated Health Reporting: Generates comprehensive stress-level and physical condition reports.

Full-Stack Management: Built with Flask and SQLAlchemy for secure and efficient record-keeping.

🛠️ Technical Stack
Backend: Python 3.12.7 / Flask

Database: MySQL (MariaDB) / SQLAlchemy

AI/Computer Vision: YOLOv5, OpenCV, PyTorch

Frontend: HTML5, CSS3, JavaScript

📁 Project Structure
The system is organized into five core diagnostic modules:

Behavior & Stress: Real-time posture and emotion tracking.

Skin & Fur: Dermatological assessment via image processing.

Parasite Detection: Identifying external parasites (ticks/fleas).

Physical Condition: Weight and body condition scoring (BCS).

Breed Identification: Automated classification for patient profiling.
